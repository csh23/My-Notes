### 使用WAMP服务器

### 功能列表
1. 默认具有“全部笔记”和“未分类”两个条目，可以创建、删除、
重命名自己的分类条目，分类层次可只实现一级分类。
2. 笔记列表：列出所选择分类下的所有笔记。可以创建新的笔记，可以选中后移
动到新的分类中。
3. 添加笔记：允许用户添加新笔记，包括笔记标题、内容和最后修改时间。
4. 编辑笔记：允许用户编辑已有笔记信息。
5. 删除笔记：允许用户删除笔记。
6. 搜索笔记：允许用户按笔记标题或内容搜索笔记。
7. 私密笔记：允许用户将笔记设为“私密”状态，通过分类中的“私密笔记”条目浏览。支持设置笔记访问密码。
8. 笔记回收站：删除的笔记移入“最近删除”分类下，业务系统中称为“假删”。 在“最近删除”分类下再次删除则真正删除数据。
9. 笔记搜索功能增强：增加全文搜索功能，可根据笔记内容、标题和标签进行检索。 支持关键词高亮显示。
10. 笔记导出：导出笔记为文本文件（Markdown格式）
11. 夜间模式：提供夜间模式，使用CSS的媒体查询和变量切换，适应不同光线条件下的用户需求。
12. 笔记排序：支持按时间、标题等进行笔记排序。
13. 拖拽排序与分类管理：使用HTML5 Drag and Drop API实现笔记分类在分类列表中的拖拽排序。 支持通过拖拽笔记到分类条目上完成分类调整。
14. 富文本编辑器：实现笔记内容的富文本编辑功能（字体加粗、字体倾斜、添加列表、插入链接），笔记内容存储为HTML 格式。
15. 多语言支持：提供界面多语言切换功能（中文、英文）。 自动根据浏览器语言显示默认语言。
16. 笔记统计功能：增加数据统计功能，显示每个分类下的笔记数量。
17. 使用服务器端程序和数据库实现数据持久化：修改数据库模拟对象的实现，在对象方法中通过Ajax方式与服务器交互，使用PHP在 服务器端实现数据的CRUD操作。
18. 多用户支持：提供注册与登录功能，支持不同用户管理各自的笔记数据。
